### Prisma with PalentScale

initial PalentScale settings
```shell
pscale login
pscale database create YOUR_DB_NAME --region ap-northeast
yarn db:push
```

change **YOUR_DB_NAME** in package.json
```shell
yarn db:login
yarn db:connect
yarn db:studio
```
